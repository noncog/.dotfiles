* Noncog's Wallpaper Collection

Because I download wallpapers from sites like Reddit or Imgur, I have no way of knowing if there is a copyright on these images. To my knowledge, these wallpapers are "fair use". If you find an image hosted in this repository that is yours and of limited use, please let me know and I will remove it.
